技能结构
========

## 岗位

- architecture(架构)
- engineering(工程化)
- framework(框架)
- language(语言)
- tutorials(教程)
- practice(实践)
- showcase(示例)

## 框架

- tutorials(教程): 官方的 API 测试 Demo
- practice(实践): 常见问题，性能优化等
- architecture(架构): 实际项目应用
- library(开源库): 第三方库的实践
- extension(扩展): 一些衍生项目
- source(源码): 源码研究
- tools(工具): 脚手架，调试等
